## :dart: About ##

Hexnity is a high-performance REST API backend built with **Go**. Designed for scalability and efficiency, this project serves as the core engine for the Hexnity Platform, providing robust data handling and seamless integration capabilities.

## :sparkles: Features ##

:heavy_check_mark: **High Performance:** Leverages Go's concurrency model for fast request handling;\
:heavy_check_mark: **Clean Architecture:** Structured for maintainability and easy testing;\
:heavy_check_mark: **JSON RESTful Endpoints:** Fully compliant with REST standards;\
:heavy_check_mark: **Middleware Support:** Integrated logging, security, and CORS handling.

## :rocket: Technologies ##

The following tools were used in this project:

- [Go](https://go.dev/) (Golang)
- [Gin Gonic](https://gin-gonic.com/) (or your preferred framework like Fiber/Echo)
- [GORM](https://gorm.io/) (ORM for database interaction)
- [PostgreSQL](https://www.postgresql.org/) (Database)
- [Docker](https://www.docker.com/)

## :white_check_mark: Requirements ##

Before starting :checkered_flag:, you need to have [Git](https://git-scm.com) and [Go](https://go.dev/dl/) (version 1.21+) installed.

## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone [https://github.com/Hexnity-Platform/hexnity](https://github.com/Hexnity-Platform/hexnity)

# Access
$ cd hexnity

# Download dependencies
$ go mod
